# Terminal Velocity
Bullets fly, flying faster, and faster you must become too.
**Terminal Velocity** is a fast-paced terminal bullet-dodging game written in Python.

## Controls
- **WASD** / **Arrow keys** to move

## Features
- Increasing difficulty
- Online leaderboard (Firestore)
- Terminal rendering with ANSI graphics
- QR code support link

## Notes about leaderboard
- Security rules in place to prevent unauthorized scores.

## Requirements
- Python 3.10+
- [keyboard](https://pypi.org/project/keyboard/)
- [requests](https://pypi.org/project/requests/)
- [colorama](https://pypi.org/project/colorama/)
- [qrcode](https://pypi.org/project/qrcode/)

## Install
1. git clone https://github.com/Id1otic/TerminalVelocity.git
2. cd TerminalVelocity
3. pip install -r requirements.txt

## Run
- python main.py
- python3 main.py
